# Music App

**Description:**

В данном проекте реализовал музыкальный проигрыватель при помощи ванильного JS, HTML и CSS. Также имеется [папка](./src/) со всеми необходимыми исходниками(изображения, иконки и музыка).

В файле [arraySongs.js](js/arraySongs.js) каждая песня представлена в виде объекта. В последующем данные объекты будут использоваться для поиска путей к той или иной песне.

В файле [app.js](js/app.js) велась разработка логики приложения.

Стояли две задачи:

- реализовать основное окно, где будет **отоброжаться текушая песня**, **название песни и её исполнитель**, **изображение к данной песне с возможностью переключения песен**, а также в верхней части окна **создать и написать логику для кнопки плей-листа**;
- **реализовать окно плей-листа**, которое изначально скрыто, но **пользователь может его вызвать по нажатию на кнопку** в верхней части основного окна; после нажатия у пользователья **должна быть возможность выбора песни из представленных в плей-листе**; **реализовать закрытие окна**.

Все задачи были решены.

Более подробно с кодом вы можете ознакомиться [в данной папке](./js/).
